Analisis de Disco:
Este programa se crea para satisfacer una tarea de la materia de Sistemas Operativos de la universidad Hispanoamericana.

Se requiere implementar una aplicación para ejemplificar la planificación del disco a nivel de sistema operativo utilizando los siguientes algoritmos:
? First In First Out (FIFO)
? Shortest Seek First (SSF)
? Ascensor (SCAN)
? Ascensor Cíclico (C-SCAN)

Requisitos de la aplicación:
? Se debe permitir ingresar una cola de peticiones para cada algoritmo, así como la posición inicial de las cabezas.
? Se debe permitir definir un tamaño de disco.
? Se debe mostrar la cantidad de desplazamientos finales para cada algoritmo.
? Se debe generar un análisis de rendimiento y comparación entre los diferentes algoritmos, esto debe estar reflejado en la documentación, se pueden tomar en cuenta aspectos como largas colas, tamaños de disco altos, cortos, etc.
? Se debe mostrar el recorrido de cola de peticiones de manera gráfica y visual, mostrando los puntos de cada nodo (posición) recorrida y la línea que muestre gráficamente como se realizan los desplazamientos.
